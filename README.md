# Monty Hall Probability Simulation (Excel)

An Excel-based mathematical simulation of the famous Monty Hall problem using Bayes' Theorem and conditional probability concepts.

## Project Overview
This project was built to mathematically simulate, test, and analyze the conditional probability outcomes of the Monty Hall problem. Through 50 trials of empirical simulation within Excel spreadsheets, the model successfully proves the theoretical success variance of choosing to switch doors versus staying with the initial choice.

## Key Features & Skills Demonstrated
* **Statistical Modelling:** Applied Bayes' Theorem and conditional probability logic to design the core simulation framework.
* **Advanced Excel Formulas:** Utilized nested logical functions (IF, AND), lookup formulas, and randomization tools (RANDBETWEEN) to simulate user choices and host actions.
* **Data Visualization & Formatting:** Implemented conditional formatting and data styling to track and present trial outcomes clearly.
* **Analytical Verification:** Structured 50 randomized trials to observe empirical probability alignment with theoretical data.

## Files in this Repository
* `Monty_Hall_Probability_Simulation.xlsx`: The core spreadsheet containing simulation sheets, logical parameters, and trial data summaries.
